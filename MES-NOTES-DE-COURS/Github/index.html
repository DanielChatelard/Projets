II. Sauvegarder

Versionner en local, et ensuite versionner sur un dépôt distant

  1. git status, pour vérifier l'état.
  2. git add . , pour ajouter les modifications dans la zone d'index.
  3. git commit -m 'message' , pour sauvegarder avec un message en local.
  4. git log , pour voir la liste des commmits éffectués.
  5. git show sha-1 , pour voir un commit via son sha-1.
  6. git push -u origin master ou main , pour envoyer sur le dépôt distant Github.
