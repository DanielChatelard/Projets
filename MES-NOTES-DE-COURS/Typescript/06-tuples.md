# <span style="color: #26B260">**tuples.**</span>

[Sommaire](./00-Sommaire.md)

Permet de typer une variable avec un tableau d'éléments fini.

```typescript
    let arr: [number[], string, number[]] = [[12], "ok", [4]];
```
