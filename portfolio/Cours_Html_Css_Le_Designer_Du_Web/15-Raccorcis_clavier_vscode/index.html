Raccourcis clavier VScode

Alt + flèche du haut, déplace la ligne sélectionné vers le haut.

Alt + flèche du bas, déplace la ligne sélectionné vers le bas.

Ctrl + H, ouvre une fenêtre pour remplacer l'élément sélectionné par un autre

Ctrl + K + C, permet de mettre la ligne sélectionnée en commentaire

Ctrl + (/:), permet de mettre la ligne sélectionnée en commentaire

F12, dans un fichier JavaScript, sur l'appel d'une fonction, renvoi à la fonction de base

Ctrl + (+ ou -), pour zoomer et dézoomer le texte

lorem250, créera 250 mots de lorem ipsus, grâce à l'extension Emmet
